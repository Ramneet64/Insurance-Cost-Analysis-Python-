# Insurance-Cost-Analysis-Python-
Identify key factors influencing insurance premium costs and support data-driven pricing strategies.

Tools & Technology: Python (Pandas, NumPy, Matplotlib), Tableau, Data Cleaning &amp; Visualization
Process:
 Cleaned, transformed, and analyzed insurance datasets to identify correlations between age, BMI, smoking habits, and
premium costs.
 Created Tableau dashboards to visually present trends, risk profiles, and cost patterns.
 Generated insights to support predictive modeling and actuarial decision-making.
Result: Helped improve premium pricing strategy and risk evaluation accuracy through data-driven insights.
